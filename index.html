Energie Jeunes

Cette page est un outil de collecte de données pour les fiches de satisfaction reçues de la part des élèves. L'utilisation de cette page est simple et rapide.

Pour commencer, saisissez les fiches de satisfaction reçues dans le formulaire en remplissant les champs requis. Vous pouvez ajouter autant de fiches de satisfaction que nécessaire. Une fois la fiche saisi, appuyer sur le bouton "Saisir la prochaine fiche" pour passer a la suivante.

Une fois que vous avez saisi toutes les fiches de satisfaction, cliquez sur le bouton "Générer un tableau Excel" pour télécharger un fichier Excel qui regroupe toutes les données entrées.
